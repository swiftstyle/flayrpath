# Flayrpath

The first-home mortgage education app from Flayr Labs. Canadian first-time buyers.

Lives at <https://path.flayr.ca>.

## Tech

React 18 + Vite. Single-page app.

## Develop locally

```bash
npm install
npm run dev
```

Opens at <http://localhost:5173>.

## Deploy

Auto-deploys to Vercel from `main` branch.
